# Audio Visualizer 

An unique audio visualizer for browser made using [Web audio api](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) and [canvas](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API).
